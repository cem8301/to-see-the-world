# To See The World
Python script to display Strava activities on one map! Can be used to see everywhere you have traveled and everywhere you would still like to go!
  
![Alt text](docs/map_screenshots/full_me.jpg 'World view screenshot of my Strava activities') 

[Real interactive output](https://cem8301.github.io/to-see-the-world/)

## Intentions:
I started a long term cycle tour in May 2023 and found other cyclists asking for route advice and friends back home asking to visualize where we had been (we went in circles, so it got confusing). I also love maps and learned I could re-vamp a previous project ([all-of-the-trails project](https://github.com/cem8301/all-of-the-trails)), to cost $0 and run completely off of my phone.

- Fun project!
- Setup to display multiple athletes at once, so you can visualize your friends travels vs your own
- Include interesting metrics! Would like to add more if there are ideas.
- Simple
    - Write and run code 100% on my phone
    - Possible to code without internet, or gather data with basic internet
    - Free to develop, no servers or hostnames
    - Runs fast
    - User friendly?? WIP
